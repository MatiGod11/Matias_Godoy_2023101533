## Cómo subir archivos al repositorio

1. Clonar el repositorio:
   git clone https://github.com/MatiGod11/Matias_Godoy_2023101533.git

2. Agregar los archivos dentro de la carpeta correspondiente.

3. Guardar los cambios:
   git add .

4. Crear un commit:
   git commit -m "Descripción del cambio realizado"

5. Subir los cambios:
   git push
